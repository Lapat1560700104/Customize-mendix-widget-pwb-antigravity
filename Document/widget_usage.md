# คู่มือการนำ Mendix Pluggable Widget ไปใช้งาน (Widget Usage Guide)

เอกสารฉบับนี้อธิบายรายละเอียดทีละขั้นตอน (Step-by-Step) ในการนำวิดเจ็ต **PWB Advanced DatePicker (v1.0.5)** ที่คอมไพล์สำเร็จแล้ว ไปติดตั้ง กำหนดค่า Properties และประยุกต์ใช้งานในโปรเจกต์ของโปรแกรม **Mendix Studio Pro**

---

## 🛠️ ขั้นตอนที่ 1: การแพ็กเกจ Widget (Bundling the Widget)

ก่อนนำ Widget ไปติดตั้งใน Mendix App คุณจำเป็นต้องทำการคอมไพล์ซอร์สโค้ด (TypeScript + React) ให้อยู่ในรูปไฟล์แพ็กเกจเดี่ยวสำเร็จรูปนามสกุล `.mpk` เสียก่อน:

1. เปิด Terminal และเข้าไปที่โฟลเดอร์โครงการ:
   ```bash
   cd pwbDatePicker
   ```
2. รันคำสั่งคอมไพล์และล้างระบบโค้ดสำหรับนำไปใช้งานจริง (Production Build):
   ```bash
   npm run release
   ```
   *(หมายเหตุ: ระบบได้ทำโครงสร้างข้ามไฟล์ autogenerated ใน `.prettierignore` แล้ว ทำให้การรันคอมไพล์จะผ่านกระบวนการตรวจโค้ดเสร็จสิ้นสมบูรณ์แบบ 100% ไร้ข้อขัดข้อง)*
3. เมื่อคอมไพล์เสร็จสิ้น ไฟล์ชิ้นงานสำหรับใช้ผลิตจริงจะอยู่ในโฟลเดอร์ `dist/` ภายใต้ชื่อ:
   `pwbDatePicker/dist/1.0.5/pwb.PwbDatePicker_1.0.5_YYYYMMDD_HHMMSS.mpk`
   และไฟล์นี้จะถูกคัดลอกส่งต่อไปยังแอป Mendix ปลายทางให้อัตโนมัติ (หากคุณตั้งค่า Project Path ไว้)

---

## 📥 ขั้นตอนที่ 2: การนำไฟล์เข้าสู่โครงการ Mendix (Importing to Mendix App)

เมื่อได้ไฟล์ `.mpk` แล้ว มี 2 วิธีในการอัปเดตตัววิดเจ็ตเข้าไปที่ระบบของ Mendix App ของคุณ:

### วิธีที่ A: การคัดลอกไฟล์โดยตรงไปยังโฟลเดอร์ Widgets (แนะนำ)
1. คัดลอกไฟล์ `.mpk` เวอร์ชันล่าสุดที่คอมไพล์ได้จากโฟลเดอร์ `dist/` ของโปรเจกต์
2. นำไปวางในโฟลเดอร์ย่อยชื่อ **`widgets`** ที่ตั้งอยู่ตรงระดับเดียวกับไฟล์โครงการ Mendix App (`.mpr`) ของคุณ:
   ```bash
   [Mendix App โฟลเดอร์หลัก]/
   ├── widgets/
   │   └── pwb.PwbDatePicker_1.0.5_20260529_145026.mpk  <-- วางไฟล์ที่นี่
   └── [ProjectName].mpr
   ```
3. เมื่อวางเสร็จแล้ว ในโปรแกรม Mendix Studio Pro ให้กดปุ่ม **F4** บนคีย์บอร์ด (หรือไปที่เมนู **App** > **Synchronize Project Directory**) เพื่อซิงค์รีเฟรชอัปเดตไฟล์

### วิธีที่ B: การอิมพอร์ตผ่านคำสั่งโปรแกรม Studio Pro
1. เปิดโปรแกรม Mendix Studio Pro
2. ไปที่แถบเมนูหลักเลือก **App** > **Import Widget Package...**
3. เลือกไฟล์ `.mpk` ที่อยู่ในเครื่องของคุณ โปรแกรมจะคัดลอกและอัปเดตในโฟลเดอร์ระบบให้เสร็จสรรพ

---

## 🖥️ ขั้นตอนที่ 3: การวางและการตั้งค่า Properties เบื้องต้น (Placing & Base Config)

1. เปิด Mendix Studio Pro และเปิดหน้าเพจ (Page) หรือโมดัลที่ต้องการวางตัวปฏิทิน
2. ในแผงแอปพลิเคชันขวาด้านบน ไปที่แถบ **Toolbox** 
3. พิมพ์ค้นหาคำว่า `"PWB Advanced DatePicker"` จากนั้นทำรายการ **ลากและวาง (Drag & Drop)** ตัววิดเจ็ตลงไปยังแถบเลย์เอาต์หน้าจอ
4. **ดับเบิลคลิก** ที่ตัววิดเจ็ตเพื่อเปิดหน้าต่างการตั้งค่า (Properties Dialog):
   * **โหมดการเลือก (Selection Mode):** สลับเลือกระหว่าง `Single Date` (ปฏิทินวันเดียว) หรือ `Date Range` (เลือกแบบช่วงเริ่มต้น-สิ้นสุด)
   * **ผูกตัวแปร Attribute:** เลือกลิงก์ฟิลด์ตัวแปรประเภท DateTime จากฐานข้อมูลของคุณ เช่น `dateAttribute` หรือ `startDateAttribute`/`endDateAttribute` ให้สัมพันธ์กับโหมดที่ระบุ

## 🎨 ขั้นตอนที่ 4: คู่มือการอัปโหลดและใช้งานไอคอนปฏิทินของตนเอง (Custom Icon Guide)

เมื่อคุณนำ Widget นี้ไปใช้ในโครงการ Mendix คุณสามารถอัพโหลดไอคอนของตัวเองขึ้นไปใช้ได้ทันทีผ่านขั้นตอนปกติของ Mendix Studio Pro:

### 1. อัพโหลดรูปภาพของตัวเอง (SVG / PNG / JPG):
* **สร้าง Image Collection** ขึ้นมาในโมดูลของแอป Mendix (หรือใช้ Image Collection ที่มีอยู่แล้ว)
* **อัพโหลดรูปไอคอนของคุณ**เข้าไปเก็บไว้ใน Image Collection นั้น (แนะนำอย่างยิ่งให้ใช้ไฟล์ `.svg` เพราะจะคมชัดและย่อขยายขนาดได้สวยงามโดยไม่แตก หรือสามารถใช้ไฟล์ `.png` แบบพื้นหลังโปร่งใส ก็ได้เช่นกัน)
* **ดับเบิลคลิกที่วิดเจ็ต PwbDatePicker** บนหน้า Page ของ Mendix จากนั้นที่หัวข้อ **Custom Calendar Icon** ในแท็บ **Aesthetics** ให้คลิกปุ่ม Select
* **เลือกรูปภาพจาก Image Collection**ที่คุณอัพโหลดไว้ได้ทันทีครับ! ตัววิดเจ็ตจะประมวลผลและปรับขนาดการแสดงผลลงไปในปุ่มกดให้สวยงามพอดีโดยอัตโนมัติ

### 2. เลือกใช้ไอคอนระบบมาตรฐาน (Glyphicon / FontAwesome):
* นอกเหนือจากการอัพโหลดรูปภาพเองแล้ว หากคุณต้องการใช้ไอคอนสัญลักษณ์ทั่วไป คุณก็สามารถเลือกจากชุดไอคอนมาตรฐานของ Mendix หรือไอคอนจาก FontAwesome ผ่านหน้าต่างเลือกไอคอนแบบปกติได้ทันทีด้วยเช่นกันครับ

## 💻 ขั้นตอนที่ 5: การรันเครื่องมือจำลอง Playground & อัพโหลดไฟล์ไอคอนจำลอง
เพื่อประโยชน์สูงสุดและการตรวจสอบ UI แบบเร่งด่วนโดยไม่ต้องเสียเวลาบูตแอป Mendix หรือคอมไพล์แพ็กเกจใหม่:

1. เปิด Terminal ในโฟลเดอร์วิดเจ็ตและรันคำสั่งจำลอง:
   ```bash
   npm run playground
   ```
2. เปิดลิงก์เบราว์เซอร์ที่เปิดขึ้นมา: `http://localhost:3000/`
3. ในแผงตั้งค่าฝั่งซ้าย ให้คลิกขยายแถบข้อที่ **4. Aesthetics**
4. ตรงฟิลด์ **"Custom Input Icon (ไอคอนกล่องข้อความจำลอง)"** ให้เลือกเป็นหัวข้อ **"Uploaded Custom Icon (ไอคอนที่อัพโหลดเอง)"**
5. จะมีช่องสำหรับคลิกอัปโหลดภาพไอคอนจำลองขึ้นมา คุณสามารถกดปุ่มและคลิกเลือกไฟล์ภาพใด ๆ จากเครื่องของคุณ (เช่น ไฟล์โลโก้ SVG, PNG, JPG)
6. ภาพไอคอนนั้นจะเข้าไปแทนที่สัญลักษณ์เดิมในหน้าจอ Canvas ปฏิทินฝั่งขวาและจัดตำแหน่งเซ็นเตอร์แสดงผลเอฟเฟกต์โฮเวอร์ให้คุณดูสัดส่วนความสวยงามได้ทันทีแบบสด ๆ!

## 🔄 ขั้นตอนที่ 6: การซิงค์และพัฒนาแบบเรียลไทม์ (Hot Reloading Dev Mode)
เมื่อคุณต้องการแก้ไขซอร์สโค้ด React หรือ CSS และต้องการผลลัพธ์วิ่งไปอัปเดตในระบบของแอป Mendix ทันทีโดยไม่ต้องคัดลอกไฟล์ `.mpk` ด้วยตัวเองในแต่ละครั้ง:

1. เปิดไฟล์ `package.json` ของวิดเจ็ต มองหาหัวข้อ `config`
2. แก้ไขพารามิเตอร์ `"projectPath"` ให้ชี้ไปยังพิกัดโฟลเดอร์โปรเจกต์ Mendix ของคุณแบบสมบูรณ์:
   ```json
   "config": {
     "projectPath": "/Users/user/Documents/Mendix/MyMendixAppProject"
   }
   ```
3. รันคำสั่ง Watch ใน Terminal:
   ```bash
   npm run dev
   ```
4. ระบบจะทำการเฝ้าสังเกตไฟล์ซอร์สโค้ดของคุณ เมื่อใดที่คุณกด Save โค้ด ระบบจะรวบรวมไฟล์อัปเดตและคัดลอกตรงไปยังโฟลเดอร์ Widgets ในแอป Mendix ให้ทันที 
5. ในโปรแกรม Mendix Studio Pro ให้กดปุ่ม **F4** เพื่อรีเฟรช และกดรันสั่งงานเพื่อทดสอบฟังก์ชันใหม่ของคุณได้ทันควันครับ!
